- `docs/index.md`
- `docs/architecture.md`
- `docs/getting-started/human-first-10-min.md`
- `docs/showcase/minimal-success-case.md`

If you need help choosing the right path, use GitHub Discussions first:

- `https://github.com/xiaojiou176-open/prooftrail/discussions`

## Bug reports

Open a GitHub issue with:

- steps to reproduce
- expected behavior
- actual behavior
- logs or screenshots when safe to share

## Security reports

Do not use public issues for vulnerabilities or leaked credentials.
Use the private path described in `SECURITY.md`.
